# 安裝步驟

## 1. 安裝

1. 複製設定：
   ```bash
   git clone https://github.com/dq042000/nvim-config.git ~/.config/nvim
   ```
2. 啟動 Neovim：
   ```bash
   nvim
   ```
3. 第一次啟動時，LazyVim 會自動安裝所需相依套件。
4. 依照提示完成額外設定。

## 常見問題

- 若遇到相依套件安裝失敗，請確認網路連線或手動安裝。
- 設定檔路徑預設為 `~/.config/nvim`。
